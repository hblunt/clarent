# Clarent - Web & Software Development

Modern web development agency website.

## Tech Stack

- **Next.js 16** - App Router with Turbopack
- **React 19** - Latest React version
- **TypeScript** - Type safety
- **Tailwind CSS v4** - CSS-first configuration
- **Framer Motion** - Animations and transitions
- **Three.js & React Three Fiber** - 3D carousel (projects page)
- **Vanta.js** - Animated hero background
- **Inter Display & UnifrakturCook** - Custom fonts

## Deployment

Digital Ocean

## Pages

- `/` - Homepage with hero, services overview, and projects sections
- `/services` - Interactive service slideshow with pricing
- `/projects` - Coming soon message (3D carousel ready to activate)
- `/contact` - Floating label contact form

## Development

```bash
npm run dev     # Start dev server
npm run build   # Build for production
npm start       # Start production server
```
